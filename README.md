# sensor-mobil-tinkercad
Buatlah rangkaian komponen arduino sensor mobil menggunakan tinkercad
dengan komponen
1. Sensor Ultrasonic HC-SR04
2. Modul LCD 2 X 16
3. Buzzer
4. Modul arduino uno
5. Dilayar LCD akan memunculkan pesan :
- Aman jika jarak >35
- Awas jika jarak >30
- Waspada jika jarak >15
